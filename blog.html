1. What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model. It is a programming interface for web documents that represents the page so that programs can change the document structure, style, and content.

2. How will you select HTML elements using DOM? Name the DOM methods.

Here are the ways to select HTML elements using DOM.

  • Selects elements with a specified class name.
  • Selects elements with a specified tag name.
  • Selects the first element that matches a specified CSS selector
  • Selects all elements that match a specified CSS selector.
